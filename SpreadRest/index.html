Operador Spread / Rest (...)

        El operador (...) se denomina operador Spread o Rest,
        dependiendo de cómo y dónde se utilice.

Spread

        Cuando se usa con cualquier iterable (como un
        array u objeto), actúa como "spread",
        introduciendo individualmente cada uno de sus
        elementos en la función

        let Pokemon = ["Pikachu", 35, 55, 40, 90];
        console.log(...Pokemon);
            Resultado: Pikachu 35 55 40 90

Rest

        El otro uso común de ... es unir una serie de
        valores en una array. En este caso, el
        operador se denomina "rest".

        let Pokemon = ["Pikachu", 35, 55, 40, 90]
        console.log(...Pokemon)
