Arrow Functions

   Arrow Functions son una forma de escribir
    funciones en ES6. Ahora se usará la palabra
    reservada const:

    const Resta = (a, b) => {
        return a - b;
       };

Arrow Functions “inline”

        Si el Arrow Function solo tiene un return, se puede
        omitir la estructura de bloque y escribir todo en la
        misma línea:
    
        const Resta = (a, b) => a - b;

Arrow Functions con Parámetro Único

        Si el Arrow Function solo tiene un return, se puede
        omitir la estructura de bloque y escribir todo en la
        misma línea:
    
        const Elevar = a => a * a;

Arrow Functions Retornando Objetos

        Si el Arrow Function solo tiene un return, se puede
        omitir la estructura de bloque y escribir todo en la
        misma línea:
    
        const Pokémon = nombre => ({ Pokémon: nombre }];
