let

    La palabra clave let te permite crear una
    variable con acceso limitado a cualquier bloque,
    que se denomina ámbito de bloque (scope).

    En lugar de usar var, que proporciona un ámbito
    de función, se recomienda usar let en ES6

const

    La palabra clave const te permite crear una
    constante con acceso limitado a cualquier
    bloque, que se denomina ámbito de bloque
    (scope).

    De esta forma, los datos que estén declarado
    con const, serán inmutables en la aplicación.
    Provocando un error si lo intentas.
