Extensiones de Objetos Literales

    
        ES6 permite declarar objetos literales con una
        sintaxis abreviada tanto para iniciar propiedades
        de variables como para definir métodos de las
        funciones.
        También permite tener índices de propiedades
        calculadas en una definición de objeto
        literal.

Destructuring Assignment

    
        La sintaxis de Asignación por Desestructuración
        (Destructuring Assignment) es una expresión que
        permite la extracción de datos, tanto de Arrays
        como de Objetos, en diferentes variables.
